# README

# Shop is a simple e-commerce type web application

```
$ rvm use 2.4.1@shop --create
$ bundle install
$ rake db:create db:migrate db:seed
$ rails s
```

### Run test
```
$ bundle exec rspec
```

### Default site administrator.
email: admin@shop.ua    
password: password

### Generate RDoc documentation
```
$ bundle exec rake rdoc
```
